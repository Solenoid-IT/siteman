# Software
Site Manager Command Line Tool (siteman)



# Requirements
1) php
2) certbot



# Install
1) `git clone https://github.com/Solenoid-IT/siteman.git ./siteman-download`
2) `cd ./siteman-download`
3) `sudo chmod +x ./install`
4) `./install`



# Uninstall
1) `sudo chmod +x ./uninstall`
2) `./uninstall`
